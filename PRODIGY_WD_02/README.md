# Precision Pro Stopwatch

A high-performance, professional-grade stopwatch web application built with React, Tailwind CSS, and standard JavaScript.

## Features

- **High Precision Timing**: Uses `requestAnimationFrame` for smooth, accurate time tracking.
- **Lap/Split Recording**: Track multiple intervals with a detailed split history.
- **Dynamic Interface**: Responsive layout that adjusts when lap data is present.
- **Dual Mode Support**: Fully optimized Light and Dark modes with custom color gradients.
- **Professional Aesthetics**: High-visibility typography and modern UI components.

## Technical Specifications

- **Language**: TypeScript / JavaScript (ES6+)
- **Framework**: React 19
- **Styling**: Tailwind CSS
- **Icons**: Font Awesome 6.4

## How to Run

1. Open `index.html` in any modern web browser.
2. The application is completely client-side and does not require a backend or external API keys.
